# Inventory Analytics Room Instructions

Use mode-specific instruction files when configuring Genie:

- **Mode A (tables-based room)** -> `instructions.tables.md`
- **Mode B (metric-views-only room)** -> `instructions.metric-views.md`

How to use:

1. Choose your setup mode in `README.genie.inventory.md`.
2. Open the matching instructions file above.
3. Copy/paste the full content into Genie room Instructions/System Prompt.

Do not use both instruction files in a single room.
